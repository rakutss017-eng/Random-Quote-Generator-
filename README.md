# Random-Quote-Generator

**Автор:** Илья Олангаев

**Описание:**  
Простое GUI-приложение для генерации случайных цитат. Возможна фильтрация по автору и теме, а также сохранение истории поколённых цитат.

**Использование:**  
- Нажмите кнопку «Сгенерировать цитату» для получения случайной цитаты.  
- Используйте поля «Автор» и «Тема» для фильтрации истории цитат.  
- Нажмите «Сбросить фильтр», чтобы восстановить весь список.

**Пример:**  
- Генерация: "Будь тем изменением, которое хочешь видеть в мире." — Махатма Ганди  
- Фильтрация по автору: введите "Ганди" и нажмите фильтр, чтобы увидеть только цитаты Ганди.
